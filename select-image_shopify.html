RKSYOストア（仮称）　デモサイト

クライアント様のWebページに、弊社からご提供するスクリプトを数行追加するだけで、画像を手軽にグッズ販売へ連携できます。
スクリプト追加後、すぐに販売がスタートできます。
専門的な知識や大きな改修は不要。
シンプルな導入で、オリジナルグッズ販売をすぐにご体験いただけます。

実装例１：画像をクリックすると販売ページが開く

写真の商品が購入できます。気に入った写真をクリックしてください。

※注意：クリックすると別サイト（RKSYOサイト（仮名））の販売ページに移動します。

実装例２：ボタンをクリックすると販売ページが開く


ウェブページへのスクリプト追加方法

導入手順はたったこれだけ。クライアント様のウェブページにあわせた導入方法について丁寧に支援いたします。

１：以下のようなスクリプトコードを契約後にご支給しますので、ウェブページに追加します

※以下のコードはサンプルです。

<!-- RKSYOストア用スクリプト：start -->
<script>
  function goToForm(imageUrl) {
    // 遷移時に画像URLとclientをGETパラメータに乗せて送る
    const client = 'とうしょう';
    const encode_client = encodeURIComponent(client);
    const encodedUrl = encodeURIComponent(imageUrl);
    window.location.href = `https://tosyoteststore5.myshopify.com/pages/custom-form.html?client=${encode_client}&image=${encodedUrl}`;
  }
</script>
<!-- RKSYOストア用スクリプト：end -->

２：商品販売ページに連携したい画像のimgタグやbuttonタグ等にonclickイベントを追加します

画像imgに追加する場合：

<img src="画像のURLまたはファイルパス" onclick="goToForm(this.src)">

buttonタグ等に追加する場合：

<button class="button" onclick="goToForm('画像のURL')">この写真のグッズを購入する</button>

【テスト】

画像URLとクライアント名（販売事業者名）に「とうしょう」を入力の上、「グッズ購入」ボタンをクリックしてみてください。

クリックすると、別サイト（RKSYOストア（仮称））に遷移し、 以下の画像の商品購入ページが開きます。
