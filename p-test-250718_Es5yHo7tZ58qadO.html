まるか食品様の模擬ページ

オリジナルパッケージメーカーの画像を使ったグッズ購入の模擬ページです。

「グッズ購入」ボタンをパッケージメーカーの完成画面に追加します。

以下の「グッズ購入」ボタンを実際にクリックしてみてください。

クリックすると、別サイト（RKSYOストア（仮称））に遷移し、 以下の画像の商品購入ページが開きます。

ウェブページへのスクリプト追加方法

導入手順はたったこれだけ。クライアント様のウェブページにあわせた導入方法について丁寧に支援いたします。

１：以下のようなスクリプトコードを契約後にご支給しますので、ウェブページに追加します

※以下のコードはサンプルです。

<!-- RKSYOストア用スクリプト：start -->
<script>
  function goToForm(imageUrl) {
    // 遷移時に画像URLとclientをGETパラメータに乗せて送る
    const client = 'まるか食品';
    const encode_client = encodeURIComponent(client);
    const encodedUrl = encodeURIComponent(imageUrl);
    window.location.href = `https://tosyoteststore5.myshopify.com/pages/custom-form.html?client=${encode_client}&image=${encodedUrl}`;
  }
</script>
<!-- RKSYOストア用スクリプト：end -->

２：buttonタグにonclickイベントを追加します

<button class="button" onclick="goToForm('画像のURL')">この写真のグッズを購入する</button>
